# Password Analyzer & Generator
This is a simple project to generate and analyze passwords.
For the first mode - Analyze, you can type in any password and it'll tell you the entropy, estimated time to crack with bruteforcing, dictionaries and suggestions to improve it. The Generator as it suggests makes a strong password.

## How to Set it up?
Clone the Repo. Install npm Packages. Do "npm run build", wait for it to finish, then do "npm run preview." It's up and running now. 

**Live Demo: https://pass-analysis.vercel.app**
